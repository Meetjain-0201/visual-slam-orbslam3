# Visual SLAM with ORB-SLAM3

Implementation and analysis of ORB-SLAM3 for monocular, stereo, and RGB-D visual SLAM.

## System
- CPU: Intel Core Ultra 7 255HX
- GPU: NVIDIA RTX 5060 8GB
- RAM: 32GB
- OS: Ubuntu 22.04 + ROS2 Humble

## Dependencies
- Pangolin v0.6
- OpenCV 4.x
- Eigen3
- g2o
- DBoW2

## Structure
- `ORB_SLAM3/` - ORB-SLAM3 source
- `datasets/` - EuRoC/KITTI datasets
- `results/` - trajectory outputs
- `scripts/` - evaluation and visualization tools
- `docs/` - notes and benchmarks
